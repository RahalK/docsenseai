# DocSenseAI

## OCR Dependency

This app uses Tesseract OCR, which is already included in the project. The `tesseract/` folder at the project root contains `tesseract.exe` and the required `tessdata/` folder:

docsenseai/
├─ tesseract/
│ ├─ tesseract.exe
│ └─ tessdata/

No additional installation of Tesseract is required.

---

## Running the App

1. Download and unzip `DocSenseAI_v1.zip`.
2. Double-click `DocsenseAI.bat`.
3. Your browser will open the app at [http://localhost:8501](http://localhost:8501).

---

## Notes

- Anaconda must be installed to run the app.  
- All necessary dependencies are included in the project folder.
