## OCR Dependency (Tesseract)

This app uses Tesseract OCR.

### Option A — Automatic (recommended)
Place the Tesseract binaries in a folder named `tesseract/` at the project root:

docsenseai/
├─ tesseract/
│  ├─ tesseract.exe
│  └─ tessdata/

### Option B — System install
Install Tesseract and make sure `tesseract.exe` is in your PATH.
